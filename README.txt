Pavimentos Chile - Sitio estático

Estructura:
- index.html
- css/styles.css
- js/main.js
- assets/img/* (imágenes de ejemplo)

Cómo reemplazar imágenes:
1) Sustituye los archivos en assets/img por tus fotos reales manteniendo los mismos nombres, o
2) Cambia las rutas dentro de index.html donde corresponda (servicio1.jpg, trabajo1.jpg, etc.).
La grilla usa 'object-fit: cover' para alinear las imágenes sin deformarlas.

Publicación rápida:
- Arrastra la carpeta a Netlify, Vercel (Static) o GitHub Pages.
- Para formulario sin backend, Netlify puede captar envíos con data-netlify="true".

Personalización:
- Cambia colores en :root en css/styles.css
- Edita textos de secciones (Servicios, Nuestros trabajos, Nosotros, Contacto).

SEO básico:
- Edita <title> y meta description en el <head> de index.html.
